## 📋 Overview

A modern, interactive web application featuring a dynamic grid system with smooth animations and a stunning purple-themed UI. Built with vanilla JavaScript, HTML5, and CSS3.

## ✨ Features

- 🎨 **Purple-themed Design** - Eye-catching gradient backgrounds and purple accents
- 🔲 **Dynamic Grid System** - Interactive 8x8 grid with hover effects
- 💫 **Smooth Animations** - Fade-in effects and transform transitions
- 📱 **Responsive Layout** - Adapts to different screen sizes
- ⚡ **Pure Vanilla JS** - No dependencies, lightweight and fast

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| HTML5 | Structure & Semantics |
| CSS3 | Styling & Animations |
| JavaScript | Interactivity & Logic |

## 🚀 Quick Start

Clone the repository: `git clone https://github.com/dvdr6/hobby.git`
Navigate to project: `cd hobby`
Open in browser: `open index.html`
OR just open github pages: `https://dvdr6.github.io/hobby`

## 📁 Project Structure

hobby/ ├── index.html (Main HTML structure) ├── styles.css (Purple-themed styling) └── script.js (Grid logic & interactions)

## 💡 Key Components

**Grid System:** 8x8 Layout with responsive design, Interactive Blocks with hover effects, Dynamic Content generation

**Animations:** Fade-in on load, Transform transitions, Smooth hover states
